HTML 5

HTML5 es la última versión del lenguaje de marcado HTML, utilizado para estructurar y presentar contenido en la web

CSS 3

Última versión del lenguaje de hojas de estilo en cascada, que define la presentación visual de documentos HTML

Javascript y DOM

Lenguaje de programación de alto nivel, interpretado, que se utiliza principalmente para agregar interacciones a páginas web.

Bootstrap

Framework de código abierto, una plantilla o conjunto de herramientas para el diseño web, que facilita la creación de sitios web responsivos y adaptables a diferentes dispositivos.

Node y Express

Entorno de ejecución de JavaScript de código abierto y multiplataforma que permite a los desarrolladores crear aplicaciones del lado del servidor y de red con JavaScript.

React

React (también llamada React.js o ReactJS) es una biblioteca Javascript de código abierto diseñada para crear interfaces de usuario con el objetivo de facilitar el desarrollo de aplicaciones en una sola página.

Git y Github

Git es Sistema de control de versiones distribuido, de código abierto, que se utiliza para rastrear cambios en archivos y coordinar el trabajo en proyectos de software. Github es una Plataforma basada en la nube donde puedes almacenar, compartir y trabajar junto con otros usuarios para escribir código.

Shell y Cli

Programa que actúa como interfaz entre el usuario y el sistema operativo, permitiendo interactuar con el sistema a través de comandos de texto.

SQL y MySQL

Lenguaje de consulta y Sistema de gestión de bases de datos relacionales de código abierto, utilizado para almacenar y gestionar datos

Wordpress

Sistema de gestión de contenidos (CMS) de código abierto y gratuito que permite a los usuarios crear y gestionar sitios web, blogs y tiendas online de manera sencilla, sin necesidad de conocimientos técnicos avanzados.
